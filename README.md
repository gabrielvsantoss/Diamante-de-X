# Diamante de X

- Neste trabalho realizei a criação de um diamante feito de X, o usuario digita um numero impar  o sistema realiza a construção de um diamante em que o meio do diamante terá a quantidade de X digitada pelo usuario


## Funcionalidades
- **Validacao de Numero Impar**: O sistema consegue validar se o usuario digitou realmente um numero impar
- **Realização de um diamante totalmente proporcional**: O sistema realiza a construção de um diamante completo.
